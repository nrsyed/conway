Click anywhere in the grid to toggle cells on/off.

Survival range: Living cells with at least the minimum and at most the maximum number of neighbors survive to the next generation.

Birth value: Dead cells with the selected number of neighbors become live cells.

Grid size: Number of rows/columns.

Delay: Milliseconds between generations.

Random density: Probability (between 0.01 and 1.0) of each cell in the grid being alive when Randomize is clicked.

Survival range and birth value can be adjusted in real time. Changes to grid size, delay, and random density take effect after clicking "Set". Changing grid size randomizes the grid.
